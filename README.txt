GITHUB UPLOAD INSTRUCTIONS
--------------------------
1. Open your GitHub repository (the one connected to dutyinsight.com).
2. Click "Add file" > "Upload files".
3. Drag ALL files from this folder into the repository root.
4. When prompted, REPLACE the existing files (index.html, style.css, sitemap.xml, etc.).
5. Commit changes directly to main.
6. Wait 30-60 seconds for GitHub Pages to redeploy.
7. Hard-refresh https://dutyinsight.com/ (Ctrl/Cmd + Shift + R) to clear cache.
